Benji Moore Historical Collection

A curated CSS theme featuring 191 classic colors inspired by authentic American architecture and design through the centuries.

How to Use This Theme

Include the CSS file in your project and use the CSS custom properties:

<link rel="stylesheet" href="benjamin-moore-historical.css"> /* In your CSS */ .my-element { background-color: var(--bm-hc-172-revere-pewter); color: var(--bm-hc-154-hale-navy); border: 1px solid var(--bm-hc-153-pewter); }
